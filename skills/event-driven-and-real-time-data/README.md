# Event-Driven and Real-Time Data

Use this category when the task is mainly about asynchronous events, CDC, streams or real-time operability.

## Start here when

- the system communicates through events rather than direct request/response;
- you need event schemas, compatibility or streaming platform design;
- the work involves CDC, windows, joins, replay, lag or back-pressure.
