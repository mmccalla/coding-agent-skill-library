# KRAG Systems Skills Manifest

Use these skills when the graph itself must support production-quality KRAG rather than acting only as a manually curated routing overlay.

| Skill | Use when |
|---|---|
| `krag-system-design` | Designing KRAG architecture, graph role, retrieval strategy, provenance model and implementation slices. |
| `krag-ingestion-graph-construction` | Building ingestion, extraction, evidence anchoring, entity resolution, graph construction and validation pipelines. |
| `krag-retrieval-answering` | Implementing graph traversal, hybrid retrieval, ranking, grounded answering and abstention. |
| `krag-evaluation-governance` | Defining KRAG-specific evaluation, observability, governance and release gates. |
