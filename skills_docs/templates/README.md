# Spec and Backlog Templates

Optional templates for specification and backlog work. Use with `spec-driven-development`, `bdd-practice`, and `planning-and-task-decomposition`.

| Template | Use for |
|---|---|
| `agile-epic-template.md` | Large initiative shaping with scope, metrics, and stakeholders |
| `agile-user-story-template.md` | User-facing stories with Given/When/Then acceptance criteria |

These templates are supplementary artefacts. They do not replace the mandatory `apply-laws-of-ai` baseline or skill operating procedures.
